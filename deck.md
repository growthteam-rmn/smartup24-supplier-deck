# Smartup24
## Единая инфраструктура B2B-заказов для поставщиков

---

## Smartup24 — единая инфраструктура B2B-заказов
Smartup24 — платформа прямых заказов между торговыми точками и поставщиками.

- Прямые заказы без участия торгового агента
- Поток заявок 24/7
- Снижение операционных затрат и ошибок
- Единый каталог и прозрачность продаж

---

## Ключевые вызовы поставщиков сегодня
- Нерентабельно обслуживать удалённые точки
- Нет актуальных остатков в торговых точках
- Дорого поддерживать KPI полевой команды
- Мерчандайзинг вручную: планограммы/отчёты нерегулярны
- Out of Stock выявляется слишком поздно
- Акции сложно запускать и оценивать централизованно
- Заказы считают вручную → ошибки и потери
- Нет единого процесса «заявка → накладная», статусы не синхронизируются
- Дубли SKU, нет стандарта фото/карточек
- Нет точной геопривязки ТТ → логистика «вслепую»
- Онбординг тормозится из-за договоров
- Нужны MML/наборы позиций для экономики отгрузки

## Как работает Smartup24 (сквозной процесс)
```mermaid
flowchart LR
  A[Поставщик: каталог, цены, MML, промо] --> B[Smartup24: единые карточки SKU + модерация]
  B --> C[Торговая точка: оформляет заказ 24/7]
  C --> D[Проверка условий: MML / доступность / лимиты]
  D --> E[Передача заявки в ERP поставщика]
  E --> F[Подтверждение / корректировка / статусы]
  F --> G[Отгрузка + накладная]
  G --> H[Передача документов в POS ТТ]
  H --> I[Приход товара без ручного ввода]
  I --> J[Аналитика: продажи, OOS, промо, активность ТТ]
